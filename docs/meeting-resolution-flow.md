# ข้อสั่งการ → สาขาทำอะไรบ้าง

---

## ฝั่ง Admin (เขต)

1. เข้า `/meeting` → แท็บ **มติ / ข้อสั่งการ**
2. กด **เพิ่มข้อสั่งการ** → กรอก title, ผู้รับผิดชอบ, กำหนดวัน
3. เลือก **"แจ้งสาขา"** ✅  
   → ระบบสร้าง action item ให้สาขาอัตโนมัติ

---

## ฝั่งสาขา (หลังถูกสั่งการ)

### ขั้น 1 — รับทราบที่ `/notify`
- เห็นการแจ้งเตือนขึ้นมา (badge แดง "ใหม่")
- กด **รับทราบ** → จุดหายไป

### ขั้น 2 — เปิด task ที่ `/action`
- เห็น card **ORD-XXX-001** ชื่อ "[มติสั่งการ] ..."
- status เริ่มต้น: **รอดำเนินการ**

### ขั้น 3 — กรอกความก้าวหน้าใน card นั้น
- คลิก card → กด **อัพเดทความก้าวหน้า**
- เลือก % → 0 / 25 / 50 / 75 / **100**
- พิมพ์รายละเอียด → กด **บันทึก**

---

## status เปลี่ยนตาม %

- **0%** → รอดำเนินการ
- **1–99%** → ระหว่างดำเนินการ  
- **100%** → แล้วเสร็จ ✅

---

## Admin เห็นผลที่ไหน

- `/meeting` แท็บ **มติ / ข้อสั่งการ** → progress bar อัปเดต
- `/directive` → traffic light สี เขียว / เหลือง / แดง ของทุกสาขา
